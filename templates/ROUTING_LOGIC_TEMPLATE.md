# Routing Logic Template

## Starting Rule

Every post starts inside the temporary campaign.

After publishing and observation, it is routed into its proper permanent lane.

## Possible Outcomes

1. Keep as platform-only test.
2. Expand into short-form series.
3. Expand into long-form essay.
4. Convert into YouTube video.
5. Route into permanent content lane.
6. Discard as low-signal experiment.
7. Create new campaign.
8. Create new permanent lane.

## Routing Questions

- What was the real subject?
- Which audience responded?
- What kind of comments appeared?
- Did the post create reusable language?
- Did it produce a strong follow-up?
- Does it belong in an existing lane?
- Does it justify a new lane?
